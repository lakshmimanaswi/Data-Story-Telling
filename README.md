# 📊 Data Storytelling App

## Project Overview

This project demonstrates the process of Data Storytelling using a Netflix-style dataset. The goal is to transform raw data into meaningful insights through data analysis and visualizations.

The project covers:

- Dataset Introduction
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Data Visualization
- Insights and Findings
- Final Conclusions and Recommendations

---

## Dataset Description

The dataset contains information about movies and TV shows, including:

| Column Name | Description |
|------------|-------------|
| show_id | Unique ID of the content |
| type | Movie or TV Show |
| title | Content title |
| country | Country of origin |
| release_year | Year of release |
| rating | Content rating |
| duration | Duration of content |
| listed_in | Genre category |

## Key Insights

- Movies constitute the majority of available content.
- Content production has increased significantly in recent years.
- Several countries contribute to content creation, indicating global expansion.
- Drama and Comedy are among the most popular genres.
- Streaming platforms continue to diversify their content libraries.

---

## Conclusion

The analysis highlights important trends in entertainment content production and distribution. Data storytelling techniques help transform raw information into actionable insights. Visualizations make it easier to identify patterns, compare categories, and communicate findings effectively.
